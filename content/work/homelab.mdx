## Overview

This repository documents the design, build process, and security experiments in my personal cybersecurity homelab. The goal is a controlled enterprise-style lab for practicing system administration, networking, detection engineering, vulnerability management, and attack simulation — documented in a structured, public-facing way.

## Current phases

1. Lab foundation and networking — complete
2. AD identity and access — complete
3. Log collection and SIEM onboarding — in progress
4. Vulnerability management workflow — planned
5. Attack simulation and detection — planned
6. Linux hardening and monitoring — planned

## Stack highlights

- Windows Server 2022, Ubuntu Server, Kali Linux on VirtualBox
- Active Directory, DNS, Group Policy
- Wazuh, Sysmon, Nmap, Wireshark, and related security tooling

The lab evolves as infrastructure and testing scenarios expand. Full write-ups live in the [Homelab repository](https://github.com/gregory-dean/Homelab).
